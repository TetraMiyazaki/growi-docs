# ファイルをアップロードする

ファイルアップロードの手順について解説します。

## アップロード可能なファイルの形式

画像ファイルをアップロードできます。画像以外のファイルアップロードする場合は以下をご覧ください。

- [ファイルアップロード設定](/ja/admin-guide/management-cookbook/app-settings.html#ファイルアップロード)
